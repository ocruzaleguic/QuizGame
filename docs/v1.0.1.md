

**1. CORRECCIÓN DE ERRORES:**

1.1. Fallo al cerrar el Quiz:

    > Error:

        Al cerrar/refrescar la página durante/al final del quiz, el Quiz no se reseteaba correctamente.

    > Motivo:

        Solo los botones "Reiniciar" y "Terminar" del Quiz estaban programados para resetear la memoria del localStorage.

    > Solución:

        Se añadió un window.addEventListener(); con el evento pageHide para eliminar quiz_index y quiz_score del localStorage en caso se cierre la pestaña o se recargue.
        En auth.js se cambió location.reload(); por showCurrentQuestion(); para evitar conflictos con el EventListener y reutilizar una función existente.


**2. MEJORAS Y OPTIMIZACIONES:**

2.1. Se han generalizado funciones, y colocado en utils.js:

    > Funciones refactorizadas:

        IsGet(key, fallback)  -> Guarda en localstorage, devuelve un valor por defecto
        IsSet(key, value)     -> Guarda en localStorage, lo vuelve json
        IsRemove(key)         -> Elimina una clave específica del localStorage  
        loadJSON(path)        -> Carga un json con fetch y lo devuelve parseado
        resetKeys(keys)       -> Elimina múltiples claves
